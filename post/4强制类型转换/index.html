4、强制类型转换(JS)

Posted by     "亿观" on Thursday, February 6, 2020

将其他的数据类型转换为Strig

  • 调用被转换数据类型的toString()方法，该方法不会影响到原变量，它会将转换的结果返回；但是null和undefine这两值没有toString()方法
var a = 123;
a = a.toString();
  • 调用String()函数，可以将null和undefine转换为String类型，
var a = 123;
a = String(a);

将其他的数据类型转换为Number

  • 使用Number()函数
    1. 如果字符串中有非数字的内容，则转换为NaN；
    2. 如果字符串是空串或空格则转换为0；
    3. 布尔值转化为数字：true–>1 false–>0；
    4. null–>0；
    5. undefine–>NaN
var a = "123";
a = Number(a);
  • 该方式专用于字符串，parseInt()把一个字符串转换为一个整数，parseFloat()把一个字符串转换为一个浮点数
var a = "123px";
a = parseInt(a);
//可以将一个字符串中的有效整数内容取出

将其他的数据类型转换为布尔值

  • 使用Boolean()函数
  • 数字—>布尔值 除了0和NaN都是true
  • 字符串—>布尔值 除了空串都是true
  • null和undefine都会转换为false
  • 对象也会转换为ture

JS中其他进制数的表示方法

  • 16进制需要以0x开头
a = 0xff;
  • 8进制需要以0开头
a = 070;//结果为56
  • 2进制需要以0b开头，但不是所有浏览器都支持
a = 0b10;
  • 类似"070"这种字符串有些浏览器会将其当成8进制，有些则当成10进制，所以可以用以下方法指定进制
a = "070";
a = parseInt(a,10);

20、DOM(JS)

Posted by     "亿观" on Thursday, February 13, 2020

DOM（Document Object Model）简介

  • 文档对象模型，通过DOM可以来任意来修改网页中各个内容
  • 文档——指网页，一个网页就是一个文档
  • 对象——指将网页中的每一个节点都转换为对象，转换完对象以后，就可以以一种纯面向对象的形式来操作网页了
  • 模型——用来表示节点和节点之间的关系，方便操作页面
  • 节点（Node）
    1. 节点是构成网页的最基本的单元，网页中的每一个部分都可以称为是一个节点
    2. 虽然都是节点，但是节点的类型却是不同的
    3. 常用的节点
      • 文档节点 （Document），代表整个网页
      • 元素节点（Element），代表网页中的标签
      • 属性节点（Attribute），代表标签中的属性
      • 文本节点（Text），代表网页中的文本内容
  • 节点的属性

DOM操作

  • 在网页中浏览器已经为我们提供了document对象，它代表的是整个网页，它是window对象的属性，可以在页面中直接使用
  • document查询方法
    1. 根据元素的id属性查询一个元素节点对象：document.getElementById(“id属性值”);
    <!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
    </head>
    <body>
        <button id="btn">我是一个按钮</button>
        <script type="text/javascript">
            //获取到button对象
            var btn=document.getElementById("btn");
            //修改按钮的文字
            btn.innerHTML = "按钮";
        </script>
    </body>
</html>
//innerHTML用于获取元素内部的HTML代码，对于自结束标签，这个属性没有意义
//如果需要读取元素节点属性，直接使用：元素.属性名，例：btn.value

      2.  根据元素的name属性值查询一组元素节点对象：document.getElementsByName("name属性值");
  3.  根据标签名来查询一组元素节点对象：document.getElementsByTagName("标签名");

事件

  • 事件指的是用户和浏览器之间的交互行为。比如：点击按钮、关闭窗口、鼠标移动等，可以为事件来绑定回调函数来响应事件
  • 绑定事件的方式：
    1. 可以在标签的事件属性中设置相应的JS代码
    <button id="btn" onclick="alert('你好');">我是一个按钮</button>

      ![QQ截图20200214154547.png](http://ww1.sinaimg.cn/large/006QQPIfly1gbvzc35sm8j30s90qe0ty.jpg)
  2. 可以通过为对象的指定事件属性设置回调函数的形式来处理事件

      <button id="btn">按钮</button>
  <script>
      var btn = document.getElementById("btn");
      btn.onclick = function(){
          alert("你好");
      };
  </script>

文档的加载

  • 浏览器在加载一个页面时，是按照自上向下的顺序加载的，加载一行执行一行
  • 如果将js代码编写到页面的上边，当代码执行时，页面中的DOM对象还没有加载，此时将会无法正常获取到DOM对象，导致DOM操作失败
  • 解决方式一：将js代码编写到body的下边
<body>
	<button id="btn">按钮</button>
	<script>
		var btn = document.getElementById("btn");
		btn.onclick = function(){
		
		};
	</script>
</body>
  • 解决方式二：将js代码编写到window.onload = function(){}中，window.onload 对应的回调函数会在整个页面加载完毕以后才执行，所以可以确保代码执行时，DOM对象已经加载完毕
<script>
	window.onload = function(){
		var btn = document.getElementById("btn");
		btn.onclick = function(){
		
		};
	};
</script>

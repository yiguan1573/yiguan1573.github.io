2、nginx安装(Nginx)

Posted by     "亿观" on Sunday, September 27, 2020

安装步骤

  1. docker search nginx 命令来查看可用版本
  2. docker pull nginx:latest 拉取官方的最新版本的镜像
  3. docker run –name nginx -p 3737:80 -d nginx将本地 3737 端口映射到容器内部的 80 端口并启动(映射配置文件启动docker run –name nginx -v /opt/nginx/default.conf:/etc/nginx/conf.d/default.conf:ro -p 3737:80 -d nginx)
  4. 在浏览器中打开

nginx配置文件

  • docker exec -it nginx /bin/bash 进入到容器
  • cd /etc/nginx 进入到容器的指定位置查看配置
  • nginx容器内默认没有vim编辑器，先安装vim
$ apt-get update
$ apt-get install vim

  • 进入配置文件

  • 注意，新版nginx配置需要在/etc/nginx/conf.d/default.conf修改

  • 根据上述文件，可以很明显的将 nginx.conf 配置文件分为三部分：

  • 第一部分：全局块

    • 配置服务器整体运行的配置指令，比如 worker_processes 1; 处理并发数的配置

  • 第二部分：events 块

    • 影响 Nginx 服务器与用户的网络连接，比如 worker_connections 1024; 支持的最大连接数为 1024

  • 第三部分：http 块

    • 这是 Nginx 服务器配置中最频繁的部分，代理、缓存和日志定义等绝大多数功能和第三方模块的配置都在这里
    • 包含两部分：http 全局块和server 块
    • http 全局块配置的指令包括文件引入、MIME-TYPE 定义、日志自定义、连接超时时间、单链接请求数上限等。

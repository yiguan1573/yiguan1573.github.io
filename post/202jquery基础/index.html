2、jQuery基础1(jQuery)

Posted by     "亿观" on Friday, April 17, 2020

入口函数

  • 写法一
$(document).ready(function(){

});
  • 写法二
$(function(){

});
  • jQuery入口函数和window.onload入口函数的区别
    • jQuery入口函数可以写多个
    • 执行时机不同: jQuery入口函数要快于window.onload
    • jq的入口函数dom加载完毕后就会执行；js的入口函数window.onload,需要加载所有的资源(dom树、图片、外部资源)才执行

$是一个函数

  • $ is not defined,就说明没有引入jQuery文件
  • jQuery文件结构
(function(){
    window.jQuery = window.$ = jQuery;
}());//其实是一个自执行函数
  • 这个自执行文件就是给window对象添加一个jQuery属性和$属性
  • $和jQuery是等价的，为一个函数
  • 如果参数传递得是一个匿名函数–>入口函数
$(function(){

});
  • 如果参数传递的是一个字符串–>选择器/创建一个标签
$('#one');
$('<div></div>');
  • 如果参数是一个dom对象，就会把dom对象转换为jQuery对象

dom对象和jQuery对象

  • dom对象
    • 原生js选择器获取到的对象
    • 特点:只能调用dom方法或者属性，不能调用jQuery的属性或者方法
  • jQuery对象
var $div1 = $('#one');
$div1.css('backgroundColor','green');
  • jQuery对象是伪数组

dom对象和jQuery对象之间得转换

  • dom对象转换成jQuery
var div1 = document.getE lementById("one") ;
var $div1=$(div1);
  • jQuery对象转换成dom对象
//1.使用下标取出来
var $divs = $('div');
var div1 = $divs[0];
//2.使用jQ方法
var div2 = $divs.get(1);

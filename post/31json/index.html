31、JSON(JS)

Posted by     "亿观" on Thursday, April 9, 2020

简介

  • JS中的对象只有JS自己认识，其他的语言都不认识
  • JSON就是一个特殊格式的字符串，这个字符串可以被任意的语言所识别，并且可以转换为任意语言中的对象，JSON在开发中主要用来数据的交互
var obj = '{"name":"孙悟空"，"age":18,"gender":"男"}';
  • JSON：JavaScript Object Notation JS对象表示法，JSON和JS对象的格式一样，只不过JSON字符串中的属性名必须加双引号
  • JSON的分类
    • 对象{}
    • 数组[]
  • JSON中允许的值
    1. 字符串
    2. 数值
    3. 布尔值
    4. null
    5. 对象（普通对象不包括函数）
    6. 数组

JSON字符串的转换

  • 在JS中，为我们提供了一个工具类，叫JSON，这个对象可以帮助我们将一个JSON转换为JS对象，也可以将一个JS对象转换为JSON
  • JSON.parse()，需要一个JSON字符串作为参数，将其转换为JS对象后返回
  • JSON. stringify()，可以将一个JS对象转换为JSON字符串，需要一个js对象作为参数，会返回一一个JSON字符串
  • JSON这个对象在IE7及以下的浏览器中不支持

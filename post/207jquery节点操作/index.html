7、jQuery节点操作(jQuery)

Posted by     "亿观" on Wednesday, April 22, 2020

创建节点

  • 原生js中创建节点:document.write();innerHTML; document.createElement ( ) ;
  • jQuery创建节点的方法html()和$()方法
  • html();
    1. 获取内容: html()方法不给参数，获取到元素的所有内容
    2. 设置内容：html()方法给参数，会把原来的内容覆盖，如果设置的内容中包含了标签，是会把标签给解析出来的
  • $();
    • 确实能创建元素,但是创建的元素只存在于内存中，如果要在页面上显示，就要追加
    • jQuery中用append()添加节点

添加节点

  • append()，作为最后一个子元素添加
  • prepend()，作为第一个子元素添加
  • before()，作为兄弟元素添加到其前面
  • after(),作为兄弟元素添加到其后面
  • appendTo()，把子元素作为父元素的最后一个子元素添加

移除节点

  • html(""),不推荐使用，有可能会造成内存泄漏，不安全
  • empty()，清空内容
  • remove()，移除某一个元素

克隆节点

  • jQuery中克隆节点:clone()，只存在于内存中，如果要在页面上显示，就应该追加到页面
  • clone()方法参数不管是true还是false，都是会克隆到后代节点
  • clone()方法参数是true表示会把事件一起克隆到，参数如果是false就不会克隆事件，不给参数默认是false

获取或设置表单元素内容**

  • 原生js是通过value属性来获取或者设置表单元素的值
  • jQuery中用val()获取或设置表单元素内容
  • val()方法，不给参数就是获取
  • val()方法，给参数就是设置

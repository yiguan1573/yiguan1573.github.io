  • 安装和使用

安装和使用

  • 安装

    前提-需要有java8以及Python2.7以上的环境

    wget http://datax-opensource.oss-cn-hangzhou.aliyuncs.com/datax.tar.gz 
tar -zxvf datax.tar.gz

  • 可能遇到的问题

    • 运行官方案例

      bin/datax.py /job/job.json

    • 直接报错，说找不到plugin,需要将/plugin/reader和/plugin/writer中带._的隐藏文件删除

      rm -rf ._*

  • mysql2mysql.json

    "job": {
        "content": [
            {
                "reader": {

6、SpringSecurity和oauth2踩坑集合（SpringSecurity）

Posted by 亿观 on Saturday, October 15, 2022

java.lang.IllegalArgumentException: There is no PasswordEncoder mapped for the id “null”

  • auth2配置授权服务报错
@Override
public void configure(ClientDetailsServiceConfigurer clients) throws Exception {
    //配置两个客户端,一个用于password认证一个用于client认证
    clients.inMemory().withClient("client_2")
            .authorizedGrantTypes("password","refresh_token")
            .accessTokenValiditySeconds(3600)
            .scopes("all")
            .secret(defaultPasswordEncoder.encode("123456"));
}

解决

@Override
public void configure(ClientDetailsServiceConfigurer clients) throws Exception {
    //配置两个客户端,一个用于password认证一个用于client认证
    clients.inMemory().withClient("client_2")
            .authorizedGrantTypes("password","refresh_token")
            .accessTokenValiditySeconds(3600)
            .scopes("all")
            .secret("{noop}123456");
}
  • There is no PasswordEncoder mapped for the id “null“ 问题的解决方法

oauth2密码模式没有返回refresh_token

  • 按照如下配置
clients.inMemory().withClient("client_2")
        .authorizedGrantTypes("password","refresh_token","authorization_code")
        .accessTokenValiditySeconds(360)
        .scopes("all")
        .refreshTokenValiditySeconds(295200)
        .secret("{noop}123456");
  • 如果还是没有返回，则需要等待access_token过期后再次请求或者删除redis中的记录

restTemplate请求报错没有请求体

  • 正确的请求
//组装请求头
HttpHeaders headers = new HttpHeaders();
headers.setContentType(MediaType.APPLICATION_FORM_URLENCODED);
//必须设置 basicAuth为对应的 clienId、 clientSecret
headers.setBasicAuth("animation", "123456");

MultiValueMap<String, String> params = new LinkedMultiValueMap<>();
params.add("username","yiguan");
params.add("password","12345");
params.add("grant_type","password");
params.add("scope","all");
HttpEntity<MultiValueMap<String, String>> entity = new HttpEntity<>(params, headers);
DataResult postForObject = restTemplate.postForObject(url, entity, DataResult.class);

oauth2刷新token报错UserDetailsService is required

  • 在授权服务器configure方法中添加UserDetailsService即可
@Override
public void configure(AuthorizationServerEndpointsConfigurer endpoints) throws Exception {
endpoints
        .tokenStore(new RedisTokenStore(redisConnectionFactory))
        .authenticationManager(authenticationManager)
        //设置userDetailsService刷新token时候会用到
        .userDetailsService(userDetailsService);
endpoints.exceptionTranslator(oauthExceptionHandler);
}

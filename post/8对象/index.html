8、对象(JS)

Posted by     "亿观" on Saturday, February 8, 2020

对象的简介

  • 对象是一种复合数据类型，在对象中可以保存多个不同数据类型的属性

  • 对象的分类

    1. 内建对象：由MS标准定义的对象，在任何的ES的实现中都可以使用，如：Math String Number等
    2. 宿主对象：由JS的运行环境提供的对象，目前来讲主要指由浏览器提供的对象。如：BOM DOM
    3. 自定义对象：由开发人员自己创建的对象

  • 创建对象

		- 方式一：
			- var obj = new Object();
		- 方式二：
			- var obj = {};
  • 向对象中添加属性，语法：对象.属性名 = 属性值;
var obj = new Object();
obj.name = "张三";
obj.age = 18;
  • 读取对象中的属性，语法：对象.属性名;
document.write(obj.age);
  • 删除对象中的属性，语法：delete 对象.属性名;
delete obj.name;

属性名与属性值

  • 对象的属性名没有任何要求，属性值也可以是任意的数据类型，如果使用特殊的属性名，不能采用.的方式操作，语法：对象[“属性名”] = 属性值;
obj["123"] = 789;
document.write(obj["123"]);
  • 使用[]去操作属性更加灵活
obj["nihao"]=789;
var n="nihao";
console.log(obj[n]);
  • 属性值可以是任意数据类型，甚至可以是对象
  • in运算符，使用in检查对象中是否含有指定属性，语法：“属性名” in 对象，如果在对象中含有该属性，则返回true，如果没有则返回false
console.log("name" in obj);

基本数据类型和引用数据类型

  • 基本数据类型：String Number Boolean Null Undefined
  • 引用数据类型：Object
  • 基本数据类型的数据，变量是直接保存的它的值，变量与变量之间是互相独立的，修改一个变量不会影响其他的变量
  • 引用数据类型的数据，变量是保存的对象的引用（内存地址），如果多个变量指向的是同一个对象，此时修改一个变量的属性，会影响其他的变量
  • JS中的变量是保存到栈内存中的，基本数据类型的值直接在栈内存中存储
  • 对象保存在堆内存，每创建一个新的对象，就会在堆内存中开辟出一个新的空间，而变量保存的对象是对象的内存地址
  • 修改对象的属性值另一个对象会受到影响，但修改对象为null，另一个对象不会受到影响
  • 比较两个引用数据类型时，比较的是对象的内存地址

对象字面量

  • 使用对象字面量，在创建对象时直接向对象中添加属性，语法如下
//使用对象字面量来创建一个对象
var obj = {
			name:"猪八戒",
			age:28,
			gender:"男"
		};

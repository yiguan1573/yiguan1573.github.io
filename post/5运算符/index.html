5、运算符(JS)

Posted by     "亿观" on Thursday, February 6, 2020

算数运算符

  • 字符串相加相当于将两个字符串拼接为一个字符串，与Java相同
  • 任何值与字符串做加法运算都会被转换为字符串后再拼串
var a = "123" + 1;//结果为1231
  • 任何值做 - * /运算时都会自动转换为Number
var a = "123" - 1;//结果为122

一元运算符

  • + 和 -都是一元运算符
  • 对于非Number类型的值，都会先转换为Number再运算
var a = "123";
a = +a;//转换为数字123
var b = 1 + +"2" +3;//结果为6

自增和自减

  • a++的值等于原变量的值（自增前的值）,++a的值等于原变量新值（自增后的值），自减同理
var a=1;
document.write(a++);//结果为1
document.write(a);//结果为2

逻辑运算符

  • 非（!）：可以利用!来将其他的数据类型转换为布尔值
  • 与（&&）：对于非布尔值进行与或运算时，都会先将其转换为布尔值，在运算，最后返回原值
    1. 如果第一个值为false，则返回第一个值
    2. 如果第一个值为true，则返回第二个值
var a = 2 && 1;//都为ture返回后面的，即1
var a = 0 && NaN;//都为false返回前面的，即0
  • 或(||)：对于非布尔值，它会将其转换为布尔值然后做运算，并返回原值
    1. 如果第一个值为true，则返回第一个值
    2. 如果第一个值为false，则返回第二个值

赋值运算符

  • *有=、-=、=、+=、/=、%=

关系运算符

  • 有>、<、<=、>=
  • 如果比较的两个值是非数值，会将其转换为Number然后再比较
  • 如果比较的两个值都是字符串，此时会比较字符串的Unicode编码，而不会转换为Number
  • 比较字符编码时是一位一位进行比较，如果两位一样则比较下一位
console.log("abc"<"b");//true

相等运算符

  • == 相等会自动对两个值进行类型转换，如果对不同的类型进行比较，会将其转换为相同的类型然后再比较
  • undefine衍生自null，所以这两个值做相等判断时会返回true
  • NaN不和任何值相等，包括他本身，可以通过isNaN()函数判断一个值是否是NaN
  • != 不等也会做自动的类型转换
  • === 全等，判断左右两个值是否全等，它和相等类似，只不过它不会进行自动的类型转换，如果两个值的类型不同，则直接返回false
  • !== 不全等，和不等类似，但是它不会进行自动的类型转换，如果两个值的类型不同，它会直接返回true

条件运算符（又名三元运算符）

  • 语法：条件表达式?语句一:语句二;
  • 执行流程：如果判断结果为true，则执行语句1，并返回执行结果；如果判断结果为false，则执行语句2，并返回执行结果
var a = 30;
var b = 20;
var max = a > b ? a : b;
